# Linux Platform Assets

This folder contains Linux-specific resources for the Voice Recognition Display application.

## Icon Files

Place your Linux application icon files here:
- `app-icon.png` - Application icon (recommended sizes: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256)
- `app-icon.svg` - Vector icon for scalability

## Desktop Entry

The application icon will be used in:
- Application launcher
- Window title bar
- Task bar/dock

## File Format Support

Linux supports various icon formats:
- PNG (recommended for raster icons)
- SVG (recommended for vector icons)
- ICO (Windows icon format, also supported)
